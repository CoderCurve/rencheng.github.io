发表于 2020-02-01 本文字数： 669 阅读时长 ≈ 2 分钟

主要记录一下Mac上安装的步骤
这里的步骤和官方的差不多，直接参考官网

安装Flutter

下载FlutterSDK

  1. 进入官网下载页面
  2. 选择操作系统
  3. 下载最新版本压缩包
  4. 解压压缩包
  5. 拷贝到家目录下Library文件夹下，文件夹名称更改为Flutter（仅供参考）

配置环境变量

  1. 配置镜像

    来到你对应Shell的配置文件进行配置。
    如果你使用默认的bash那么配置~/.bash_profile
    如果你使用zsh (Mac新 系统默认是zsh)那么配置~/.zshrc

  2. 配置Flutter环境变量（将Flutter命令行工具的路径配置到Shell的配置文件中）

快捷键

快捷键 备注
Ctrl + option + O 删除未使用的import
option + Enter 自动import未导入的文件
Shift + F6 重命名（如果是Touch Bar就是Shift + Rename…）
CMD + -/+ 折起/展开代码块
CMD + . 折起/展开选中代码
CMD + , 进入设置页面
CMD + [ 光标回到上一次编辑的位置
CMD + ] 光标回到下一次编辑的位置
CMD + L 定位某一行，甚至某一个字符
CMD + / 注释
CMD + Y 查看选中类的属性
CMD + O 快速打开（一般用于快速打开某个文件）
option + Enter 扩展功能(很重要！)
Option + up(↑) 选中上一层代码（比如选中当前光标的单词，选中当前小部件的所有代码，自己试~）
CMD + option + M 将选中代码提取到某个方法中
CMD + option + L 格式化代码
CMD + Shift + -/+ 折起/展开所有代码块
Option + Shift + Up/Down 上下移动行
Command + Shift + Up/Down 上下移动方法

查看小部件源码

  • CMD + Click(鼠标左键)
  • CMD + B
  • CMD + Down（小键盘↓）
  • F4（非Touch Bar键盘）
